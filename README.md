# 🐱 Random Cat Facts

A simple web application that fetches and displays random cat facts using the **Cat Facts API** and **Axios**.

## 🚀 Features

* Fetches a new random cat fact with a button click.
* Uses **Axios** to make API requests.
* Implements **Async/Await** for handling asynchronous operations.
* Displays an error message if the API request fails.

## 🛠️ Technologies Used

* HTML5
* JavaScript 
* Axios
* Cat Facts API

## 📁 Project Structure

```text
.
├── index.html
├── app.js
```

## ▶️ How to Run

1. Clone this repository:

   ```bash
   git clone https://github.com/anmisha-dash/random-cat-facts.git
   ```

2. Open the project folder.

3. Open `index.html` in your browser.

4. Click the **"Show New Facts"** button to generate a random cat fact.

## 🌐 API Used

* **Cat Facts API:** https://catfact.ninja/fact

## 📸 Preview

* Click the button.
* A random cat fact appears below it.
* Click again to get another fact.
